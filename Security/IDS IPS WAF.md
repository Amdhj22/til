## 개요

네트워크 보안 솔루션 3종: 방화벽, IDS/IPS, WAF.

## 방화벽 진화

| 종류 | 동작 | 한계 |
|------|------|------|
| **패킷 필터** | L3/L4 IP & Port 제어 | 헤더 조작/payload 대응 불가 |
| **스테이트풀** | 세션 단위 검사 | 근본적 한계 여전 |
| **프록시** | 포워드(클라이언트 은닉)/리버스(서버 은닉) | 부하 발생 |

## IDS/IPS 탐지 방식

| 방식 | 특징 |
|------|------|
| **시그니처** | DB 패턴 매칭, 미등록 패턴 대응 불가 |
| **휴리스틱** | 통계적 이상 탐지, 오탐 가능 |
| **룰 기반** | 직접 규칙 정의 |

## IDS/IPS vs WAF

| | IDS/IPS | WAF |
|---|---|---|
| 계층 | L3~L7 | **L7만** |
| 암호화 트래픽 | 탐지 불가 | **복호화 후 검사** |

> WAF는 암호화된 payload도 복호화하여 공격 코드 탐지 가능.
