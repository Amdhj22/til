## 개요

서비스 간 통신 방식 비교: IPC → Socket → RPC → REST/gRPC.

## 통신 방식 진화

| 방식 | 특징 | 한계 |
|------|------|------|
| **IPC** | 프로세스 간 통신 (메모리 비공유) | 같은 머신 내 |
| **Socket** | L7에서 L4(TCP/UDP) 이용 | 통신 과정 직접 구현 |
| **RPC** | 원격 함수 호출, IDL로 다중 언어 | Socket 한계 극복 |

## Stub & Marshalling

```
Client → Client Stub → Marshalling → Network → Server Stub → Unmarshalling → Server
```

## 현대적 구현

| 방식 | 프로토콜 | 데이터 | 적합 케이스 |
|------|---------|--------|-----------|
| **REST** | HTTP/1.1 | JSON | 외부 API |
| **gRPC** | HTTP/2 | Protobuf | 내부 서비스 간 |
| **MQ** | AMQP 등 | 다양 | 비동기/이벤트 |
